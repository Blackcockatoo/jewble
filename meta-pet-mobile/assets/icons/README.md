# App Icons

Place your app icon assets here:

## Required Files

- **icon.png** (1024x1024) - Main app icon for iOS and base Android
- **adaptive-icon.png** (1024x1024) - Android adaptive icon foreground
- **favicon.png** (48x48) - Web favicon

## Design Specifications

### Main Icon (icon.png)

- Size: 1024x1024 pixels
- Format: PNG with transparency
- Design: Seed of Life sacred geometry in gold (#D4AF37) on black (#0A0A0A)
- Style: Minimalist, high contrast, centered
- Safe area: Keep important elements within inner 80% to account for rounding

### Adaptive Icon (adaptive-icon.png)

- Size: 1024x1024 pixels
- Format: PNG with transparency
- Foreground: Sacred geometry symbol only (transparent background)
- Safe area: Keep within inner 66% (Google's safe zone)
- The background color is set in app.json to #0A0A0A

### Favicon (favicon.png)

- Size: 48x48 pixels
- Format: PNG
- Design: Simplified version of main icon
- High contrast for visibility at small sizes

## Quick Placeholder

For development, you can use a simple geometric shape:

1. Create a 1024x1024 black square
2. Draw a gold circle or hexagon in the center
3. Save as PNG
4. Resize to 48x48 for favicon

Use online tools like:
- https://www.favicon-generator.org/ for favicons
- https://appicon.co/ for app icons
- https://makeappicon.com/ for all sizes
